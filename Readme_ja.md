# Keyboard Mouser AHK

このソフトウェア(AHK Script)で、次の操作をキーボードから行うことができます

- 再帰的に画面を分割していくマップ的なマウスカーソル移動
- ４方向のマウスカーソル移動
- クリック動作

キー設定等はカスタマイズ可能
