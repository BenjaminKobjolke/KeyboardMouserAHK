Keyboard Mouser AHK
====

Overview
このソフトウェア(AHK Script)で、次の操作をキーボードから行うことができます

- 再帰的に画面を分割していくマップ的移動
- ４方向のカーソル移動
- クリック動作

キー設定等はカスタマイズ可能
